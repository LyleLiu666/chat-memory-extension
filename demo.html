LLaMA Factory 区分训练集和测试集的方法

LLaMA Factory 支持多种方式来区分训练集和测试集:

方法 1: 使用 split 参数(推荐)

在 dataset_info.json 中添加 split 字段:

json
复制
{
  "qwen_train_data": {
    "file_name": "training_data.jsonl",
    "formatting": "sharegpt",
    "columns": {
      "messages": "messages"
    },
    "tags": {
      "role_tag": "role",
      "content_tag": "content",
      "user_tag": "user",
      "assistant_tag": "assistant",
      "system_tag": "system"
    },
    "split": "train"  // 指定为训练集
  },
  "qwen_eval_data": {
    "file_name": "eval_data.jsonl",
    "formatting": "sharegpt",
    "columns": {
      "messages": "messages"
    },
    "tags": {
      "role_tag": "role",
      "content_tag": "content",
      "user_tag": "user",
      "assistant_tag": "assistant",
      "system_tag": "system"
    },
    "split": "validation"  // 指定为验证集
  }
}

方法 2: 使用 HuggingFace 数据集格式

如果使用 HuggingFace 数据集,可以直接指定 split:

json
复制
{
  "my_dataset": {
    "hf_hub_url": "username/dataset_name",
    "split": "train"  // 或 "validation", "test"
  }
}

方法 3: 在训练配置中指定

在训练脚本或配置文件中:

yaml
复制
# train_config.yaml
dataset: qwen_train_data  # 训练时使用
val_dataset: qwen_eval_data  # 验证时使用

或者在命令行中:

bash
复制
llamafactory-cli train \
  --dataset qwen_train_data \
  --val_dataset qwen_eval_data \
  --eval_strategy steps \
  --eval_steps 100

方法 4: 自动划分数据集

如果只有一个数据文件,可以使用 val_size 参数自动划分:

yaml
复制
dataset: qwen_train_data
val_size: 0.1  # 使用 10% 的数据作为验证集

完整示例

json
复制
{
  "qwen_full_data": {
    "file_name": "all_data.jsonl",
    "formatting": "sharegpt",
    "columns": {
      "messages": "messages"
    },
    "tags": {
      "role_tag": "role",
      "content_tag": "content",
      "user_tag": "user",
      "assistant_tag": "assistant",
      "system_tag": "system"
    }
  }
}

训练配置:

yaml
复制
dataset: qwen_full_data
val_size: 0.1
do_eval: true
evaluation_strategy: steps
eval_steps: 500

这样 LLaMA Factory 会自动将数据集划分为 90% 训练集和 10% 验证集。
